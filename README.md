Health Tracker Web App

A complete Health & Wellness Tracker that helps users calculate important health metrics like BMI, BMR, and Daily Water Intake.
This project is designed to promote a healthier lifestyle through simple and interactive tools.

🚀 Features
✅ BMI Calculator
Calculates Body Mass Index
Provides health category (Underweight, Normal, Overweight, Obese)
✅ BMR Calculator
Calculates Basal Metabolic Rate
Shows Total Daily Energy Expenditure (TDEE)
✅ Water Intake Calculator
Suggests daily water intake based on weight and activity level
✅ Health Tips Section
Practical advice for fitness, nutrition, hydration, and mental wellness
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: Custom CSS
Logic: Vanilla JavaScript



How It Works
🔹 BMI Formula

BMI = Weight (kg) / (Height (m))²

🔹 BMR Formula

Calculated based on:

Height
Weight
Age
Gender
🔹 Water Intake

Based on:

Body weight
Activity level
